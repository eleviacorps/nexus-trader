# Nexus Old Archive

This folder stores files kept for reference but not part of the active runtime path.

## Layout

- `scripts/` legacy one-off helper scripts
- `artifacts/` stale local artifacts and crash/debug residue
- `docs/archive/` old master summaries
- `docs/sessions/` session transcripts/notes
- `docs/versions/v24/` V24 implementation notes and summaries
- `docs/versions/v26/` V26 summary notes
